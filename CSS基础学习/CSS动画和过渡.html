transition · animation · keyframes — 清晰对比，平滑体验

⏳ 过渡 (transition)

transform: translateY() + 过渡
#translate-demo {
  transition: transform 0.3s ease;
}
#translate-demo:hover {
  transform: translateY(-15px);
}
悬停↑

✅ 添加 transition 后，悬停时平滑上移。

transition-property 对比
/* 左：transition-property: all */
#demo-all { transition: all 0.5s; }
#demo-all:hover { background: #f97316; width: 120px; transform: rotate(10deg); }

/* 右：transition-property: background-color */
#demo-bg-only { transition: background-color 0.5s; }
#demo-bg-only:hover { background: #ef4444; width: 120px; transform: rotate(10deg); }
all
bg only

👉 左边所有变化平滑过渡；右边只有背景色过渡，宽度和旋转瞬间变化。

🔄 关键帧动画 (animation + keyframes)

@keyframes bounce
@keyframes bounce {
  0%, 100% { transform: translateY(0); }
  50% { transform: translateY(-30px); }
}
.animation-bounce {
  animation: bounce 1s infinite ease-in-out;
}
弹跳

⚡ 无限循环弹跳动画，使用 ease-in-out 缓动。

animation 简写属性
animation: name duration timing-function delay iteration-count direction fill-mode; 
animation: bounce 1s ease 0.5s infinite alternate;

→ 延迟0.5s开始，交替反向，无限次

📋 动画属性组合详解

animation: name duration timing-function delay iteration-count direction fill-mode;

✨ 常用示例：animation: bounce 1s ease 0.5s infinite alternate;
表示弹跳动画持续1秒，缓动，延迟0.5秒，无限次播放，交替方向。
