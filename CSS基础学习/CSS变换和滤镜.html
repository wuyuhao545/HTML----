🔄 CSS 变换与滤镜

transform · filter · 2D/3D变换 · 视觉效果

📐 变换 (transform)

rotate(45deg)
transform: rotate(45deg);
scale(1.5)
transform: scale(1.5);
translate(20px, -10px)
transform: translate(20px, -10px);
组合变换
transform: rotate(45deg) scale(1.2) translateX(20px);
Hover

🌈 滤镜 (filter)

blur(4px)
filter: blur(4px);
brightness(1.5)
filter: brightness(1.5);
grayscale(100%)
filter: grayscale(100%);
组合滤镜
filter: blur(2px) brightness(1.2);
滤镜
⚡ 变换与滤镜可创造丰富的视觉效果，但注意性能影响。
