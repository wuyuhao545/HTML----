🧪 CSS 变量与函数

自定义属性 · calc() · clamp() · var()

📦 自定义属性 (变量)

--primary-color: #3b82f6;
:root { --primary-color: #3b82f6; } .box { background: var(--primary-color); }
使用变量的背景

🧮 calc() 计算函数

calc(100% - 40px)
width: calc(100% - 40px); height: calc(100px + 2*20px);
宽度动态计算

⚖️ clamp() 范围限制

clamp(1rem, 5vw, 2rem)
font-size: clamp(1rem, 5vw, 2rem);
调整窗口宽度，字体大小在1rem~2rem间变化

🔧 其他函数

  • min() – 取最小值，例如 width: min(300px, 80%);
  • max() – 取最大值
  • attr() – 获取HTML属性值（需配合content使用）
⚡ CSS 变量和函数让样式更灵活、可维护，是现代CSS的必备特性。
