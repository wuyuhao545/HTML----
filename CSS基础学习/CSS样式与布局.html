图标库 Font Awesome

按钮样式优化
<button class="btn-plain">原始按钮</button>
<button class="btn-optimized">优化按钮</button>

优化后：背景色、圆角、悬停缩放、过渡，提升可见性和交互体验。

CSS 组织
<link rel="stylesheet" href="style.css">

这段文字样式来自外部 style.css（假设定义了浅灰背景和蓝色边框）。

将样式提取到外部文件，便于维护、复用，减少页面体积，避免内联样式混乱。

选择器优先级 & .sr-only
/* 错误：未定义 .sr-only，文字可见 */
<span class="sr-only-missing">隐藏文字(可见)</span>

/* 正确：定义 .sr-only-fixed */
<span class="sr-only-fixed">隐藏文字(不可见)</span>
❌ 我本应隐藏，但 .sr-only 未定义样式，所以可见
✅ 我真正隐藏了（屏幕阅读器可读） 后面文字正常

优先级示例：

这个盒子同时有类 .external-style 和 ID #special-box。
ID 选择器优先级更高，所以背景为黄色（内部样式定义），而非类定义的灰色。

排查样式未生效：检查选择器优先级、样式是否被覆盖、类名是否正确定义（如 .sr-only 缺失）。

⚡ 文本与字体控制可读性和美感，组合使用能产生丰富排版效果。
