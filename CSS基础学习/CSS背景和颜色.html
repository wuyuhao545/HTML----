🖼️ 图片布局示例

图片居中 (flex)
display: flex;
justify-content: center;
align-items: center;
按钮浮于图片
position: relative; (父)
position: absolute; (按钮)
top/left/transform居中

💡 左侧卡片：Flexbox 实现图片在容器内完全居中；右侧卡片：相对/绝对定位使按钮浮于图片上方，并通过 transform: translate(-50%, -50%) 精确居中。

🧩 背景图片填充容器

background-size: cover
background-repeat: no-repeat;
background-size: cover;

图片裁剪，但填满容器

background-size: contain
background-repeat: no-repeat;
background-size: contain;
background-color: #f1f5f9;

完整显示，左右留白

background-size: 100% 100%
background-repeat: no-repeat;
background-size: 100% 100%;

拉伸填满，可能变形

⚠️ 注意：cover 和 contain 都会保持图片原始比例，区别在于是否裁剪或留白。你可以根据实际需求选择。

💡 只有定位元素（position 非 static）的 z-index 才生效。默认层叠顺序：定位元素 > 浮动 > 普通流。同一堆叠上下文中，z-index 值越大越靠前。
