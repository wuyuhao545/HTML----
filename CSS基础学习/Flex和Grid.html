CSS Flex · Grid 完全演示

一维弹性 + 二维网格 · 容器/项目属性 · 函数/区域 · 代码+效果+注解

📐 布局双雄 — Flexbox 专注一维排列（行或列），适合组件内部分布；Grid 负责二维布局（行与列同时控制），适合页面宏观架构。下方分别详解二者属性，并配有直观示例。
