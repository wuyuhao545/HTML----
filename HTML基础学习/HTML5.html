🔁 HTML5 vs 旧版HTML

核心差异对比 · 代码示例 + 实际效果

📄 文档类型声明

HTML 4.01

<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN" "http://www.w3.org/TR/html4/strict.dtd">

冗长且难以记忆

HTML5

<!DOCTYPE html>

简洁，不区分大小写，触发标准模式

✅ 当前页面顶部使用的就是 <!DOCTYPE html>，所有现代浏览器均支持。

🏷️ 语义化标签

HTML4 使用大量 div

<div class="header">页眉</div> <div class="nav">导航</div> <div class="main"> <div class="article">文章</div> <div class="aside">侧边栏</div> </div> <div class="footer">页脚</div>

导航

HTML5 语义化标签

<header>页眉</header> <nav>导航</nav> <main> <article>文章</article> <aside>侧边栏</aside> </main> <footer>页脚</footer>
页眉
导航
文章
侧边栏
页脚

语义化标签更清晰，利于SEO和屏幕阅读器，但渲染效果与div无异。

📝 表单控件与属性

HTML4 基本控件

<input type="text" name="name"> <input type="password"> <input type="checkbox"> <input type="radio"> <textarea></textarea> <select><option>...</option></select>





HTML5 新增类型与属性

<input type="email" placeholder="邮箱" required> <input type="url"> <input type="tel"> <input type="number" min="1" max="10"> <input type="range"> <input type="color"> <input type="date"> <input type="time"> <input type="search"> <input type="file" multiple> <input type="text" pattern="[A-Za-z]{3}" title="三个字母"> <input list="browsers"> <datalist id="browsers"><option value="Chrome">...</datalist>











HTML5 新增多种输入类型和内建验证（如 required、pattern），减少JavaScript依赖。

🎵 多媒体 (audio/video)

HTML4 依赖第三方插件

<object data="movie.swf" type="application/x-shockwave-flash"> <param name="movie" value="movie.swf"> </object> <embed src="audio.mp3" type="audio/mpeg">

需要Flash或其他插件，现已不推荐。

HTML5 原生支持

<audio controls> <source src="https://www.w3schools.com/html/horse.mp3" type="audio/mpeg"> </audio> <video controls width="250"> <source src="https://www.w3schools.com/html/mov_bbb.mp4" type="video/mp4"> </video>

音频/视频无需插件，支持多种格式，提供完整JavaScript API。

🎨 Canvas 绘图

HTML4 无原生绘图

通常使用 Flash、VML 或复杂的 SVG 脚本。

没有简单的方式动态绘制图形。

HTML5 <canvas>

<canvas id="myCanvas" width="200" height="100"></canvas> <script> var ctx = document.getElementById('myCanvas').getContext('2d'); ctx.fillStyle = '#3b82f6'; ctx.fillRect(10,10,80,60); ctx.fillStyle = '#f97316'; ctx.beginPath(); ctx.arc(140,50,30,0,2*Math.PI); ctx.fill(); </script>

💾 本地存储 (localStorage)

HTML4 仅Cookie

document.cookie = "key=value; expires=...";

存储大小受限（约4KB），每次请求都会发送到服务器。

HTML5 localStorage / sessionStorage

// 存储 localStorage.setItem('name', '张三'); // 读取 localStorage.getItem('name'); // 删除 localStorage.removeItem('name');

存储容量更大（5MB+），仅客户端保存，不发送到服务器。

🗑️ HTML5 中移除的旧元素

已移除

  • <font> – 改用CSS
  • <center> – 改用 text-align
  • <big> – 改用 font-size
  • <strike> – 改用 text-decoration
  • <frame> <frameset> – 改用 iframe 或布局

现代替代

<span style="color:red;">文本</span> <div style="text-align:center;">居中</div>
CSS 控制颜色
CSS 居中
