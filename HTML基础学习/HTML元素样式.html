🧱 block · inline · inline-block 基础对比

三种核心 display 值的独立演示与直观对比，外加其他常用取值

📦 块元素 display: block

🔹 特性
  • 独占一行，自动换行
  • 默认宽度 100% 父容器
  • 可设置宽高、内外边距
  • 常见：div, p, h1, section
<div> 示例
<div class="demo-box">块元素</div>
<p class="demo-box">另一个块</p>
块元素 div

块元素 p

📄 行内元素 display: inline

🔹 特性
  • 与其他行内元素共享一行
  • 宽度由内容撑开，不可设宽高
  • 上下内边距不影响行高
  • 常见：span, a, strong, em
<span> <a> 示例
<span class="inline-demo">span</span>
<a href="#" class="inline-demo">链接</a>
span 链接 强调

🧩 行内块元素 display: inline-block

🔹 特性
  • 同行显示，但可设宽高
  • 默认宽度由内容撑开
  • 上下内外边距有效
  • 常见：img, button, input 或 CSS 设置
inline-block 示例
<div class="inline-block-demo">盒子1</div>
<div class="inline-block-demo">盒子2</div>
