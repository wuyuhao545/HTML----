📄 文档基础 <!DOCTYPE> <html> <head> <body>
<!DOCTYPE html>文档类型声明
<!DOCTYPE html>
作用说明
告知浏览器采用HTML5标准渲染页面。本页面第一行即为此声明，确保现代标准模式。
<html>HTML根元素
<html lang="zh"> ... </html>
实际效果
整个页面内容都被包含在<html>中，lang="zh" 有助于屏幕阅读器和搜索引擎。当前页面即如此。
<head>文档头部
<head> <meta charset="UTF-8"> <title>页面标题</title> ... </head>
实际效果
头部内容不会直接显示，但定义了元数据、样式、脚本等。当前页面的样式就是在<head>中定义的。
<body>文档主体
<body> 所有可见内容... </body>
实际效果
你现在看到的所有卡片、文字、按钮都在<body>内。每个页面只有一个<body>。
🏷️ 头部元数据 <title> <meta> <link> <style> <script> <base> <noscript>
<title>页面标题
<title>页面标题示例</title>
实际效果
当前浏览器标签页/窗口标题为： (由<title>设置)
<meta charset>字符编码
<meta charset="UTF-8">
效果说明
声明文档使用UTF-8编码，确保中文、特殊符号正常显示。当前页面头部已设置，所有文字均正确渲染。
<meta name="viewport">视口设置
<meta name="viewport" content="width=device-width, initial-scale=1.0">
效果说明
使页面在移动端合理缩放，width=device-width 适配设备宽度。当前页面已设置，可尝试在手机查看。
<meta> 描述/关键词/作者SEO 元数据
<meta name="description" content="HTML基础框架演示"> <meta name="keywords" content="HTML, 框架, 标签"> <meta name="author" content="示例">
效果说明
这些meta提供页面描述、关键词和作者信息，供搜索引擎和浏览器使用。当前页面实际包含以下meta：
  • 描述: HTML基础框架标签大全 · 代码与效果展示 (本页)
  • 视口: 已设置
<link>链接外部资源
<link rel="stylesheet" href="style.css"> <link rel="icon" href="favicon.ico">
模拟效果
下方小方块背景色通过内部样式模拟：实际link可加载CSS、图标等。
⬅️ 此颜色由<style>定义，但<link>也能做到。
<style>内嵌样式
<style> .demo { color: #2563eb; } </style>
实际效果
这段文字的背景和边框由<style>定义（位于head内）。整个页面样式均由<style>控制。
<script>脚本
<script> console.log("Hello"); // 或定义函数 </script>
实际效果

页面上方也有一段内联脚本用于显示标题，证明<script>已执行。

<noscript>无脚本内容
<noscript>您的浏览器未启用JavaScript</noscript>
模拟效果
✔️ 脚本已启用，你看不到noscript内容。
<base>基础路径
<base href="https://example.com/" target="_blank">
作用说明
定义页面中所有相对URL的基准地址。当前页面未设置，因此相对链接相对于当前目录。若设置，所有相对链接都会受影响。
