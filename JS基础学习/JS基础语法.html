📌 变量声明 (var / let / const 特点)

var
var name = "张三";
🔹 特点：
  • 函数作用域 / 全局作用域
  • 存在变量提升（hoisting）
  • 可重复声明同一变量
  • 全局作用域下成为 window 属性
let
let age = 25;
🔹 特点：
  • 块级作用域 { }
  • 暂时性死区（TDZ），无变量提升
  • 同一作用域内不可重复声明
  • 全局声明的变量不属于 window
const
const PI = 3.14;
🔹 特点：
  • 块级作用域，必须初始化赋值
  • 常量引用不可变（对象属性可变）
  • 不存在变量提升，有暂时性死区
  • 不可重复声明

🔢 数据类型 (原始类型 + 引用类型)

String
let str = "Hello";
📝 特点： 表示文本；可用引号 ''、""、``；不可变；有 length 和丰富方法。
Number
let num = 42;
🔢 特点： 整数和浮点数；包含 Infinity、-Infinity、NaN；双精度64位。
Boolean
let bool = true;
✅ 特点： 仅 true 和 false；常用于条件判断；逻辑运算符返回布尔值,默认是false。
null
let a = null;
🚫 特点： 表示“空”或“不存在”；typeof null 返回 "object"（历史遗留）；主动赋值空缺。
undefined
let b;
❓ 特点： 变量声明未赋值；函数默认返回值；typeof undefined 返回 "undefined"。
Symbol
let sym = Symbol();
✨ 特点： ES6新增；每个 Symbol() 独一无二；常用于对象属性键避免冲突。
Object
let user = { name: "Alice", age: 25 };
📦 特点： 键值对集合；引用传递；动态增删属性；基于原型链。
Array
let colors = ["red", "green", "blue"];
📋 特点： 有序列表；索引从0开始；动态长度；继承数组方法。
Function
function greet() { return "Hello"; }
⚡ 特点： 可调用对象；多种定义方式；一等公民；形成闭包。

🧮 运算符

算术运算符
10 + 5 = 15
10 - 5 = 5
10 * 5 = 50
10 / 5 = 2
10 % 3 = 1
比较运算符
5 == '5' → true
5 === '5' → false
5 > 3 → true
5 <= 5 → true
逻辑运算符
true && false → false
true || false → true
!true → false
typeof 运算符
typeof 42 // "number"
typeof "hello" // "string"
typeof true // "boolean"
typeof undefined // "undefined"
typeof null // "object"
typeof {} // "object"
typeof [] // "object"
typeof function(){} // "function"
🔍 特点：
  • 返回表示数据类型的字符串
  • 对 null 返回 "object"（语言缺陷）
  • 函数返回 "function"
  • 数组、普通对象等都返回 "object"
  • 未声明变量返回 "undefined" 不报错
三元运算符 (?:)
条件 ? 表达式1 : 表达式2
示例：
(5 > 3) ? '大于' : '小于或等于' → '大于'
(2 === '2') ? '相等' : '不相等' → '不相等'
❓ 特点：
  • JS中唯一的三元运算符
  • 根据条件返回其中一个值
  • 可嵌套但降低可读性
  • 简化简单if...else

📌 变量提升与闭包 (Hoisting & Closure)

函数声明提升
// 可以在声明前调用 sayHi(); function sayHi() { return "Hello"; }
var 变量提升
console.log(a); // undefined var a = 5; console.log(a); // 5
let/const 暂时性死区
// 不能提前访问 console.log(b); // ReferenceError let b = 10;
闭包基本示例
function outer(x) { return function(y) { return x + y; }; } const add5 = outer(5);
内部函数记住外部作用域的变量。
计数器闭包
function createCounter() { let count = 0; return function() { return ++count; }; } const counter = createCounter();
⚡ 基础语法是JS的基石，函数是一等公民。变量提升与闭包现已独立成章，结构更清晰。本页汇集两大核心，示例均可执行。
