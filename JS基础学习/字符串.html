基础 创建 & 访问
[] 字面量
数组字面量
最常用、最简洁的创建方式。
let arr = [1, 'a', true];
let empty = [];
[1, "a", true] ; []
Array() 构造器
Array 构造函数
单个数字参数表示长度，多个参数表示元素。
let arr1 = new Array(3);
let arr2 = new Array(1,2,3);
[空 × 3] ; [1,2,3]
Array.of() / from()
ES6 创建方式
of 避免单数字歧义；from 从类数组创建。
Array.of(3);        // [3]
Array.from('hi');   // ['h','i']
[3] ; ["h","i"]
.length
数组长度
可读可写，修改 length 可截断或扩展数组。
let arr = [1,2,3];
arr.length = 2;
arr.length = 4;
[1,2] ; [1,2,空×2]
陷阱 typeof 与数组判断
typeof []
typeof 的“谎言”
在 JavaScript 中，typeof 将数组视为 "object"，因为数组是特殊的对象。这常常导致判断失误。 
console.log(typeof [1,2,3]); // "object"
console.log(typeof {});       // "object"
console.log(typeof null);     // "object"
控制台输出: "object" ; "object" ; "object"
✅ 正确判断数组：Array.isArray([]) → true
✅ [] instanceof Array → true (但跨 frame 可能有问题)
✅ Object.prototype.toString.call([]) → "[object Array]"
Array.isArray() 正确方法
判断数组
静态方法，专门用于检查变量是否为数组，返回布尔值。
Array.isArray([]);    // true
Array.isArray({});    // false
Array.isArray('hello');// false
true ; false ; false
方法 添加 & 删除 (栈/队列)
push() / pop()
末尾操作
push 尾部添加，返回新长度；pop 移除最后一项，返回该项。
let arr = [1,2];
arr.push(3,4);
arr.pop();
push后→[1,2,3,4] ; pop后→[1,2,3] 弹出4
unshift() / shift()
开头操作
unshift 头部添加，shift 移除第一项并返回。
let arr = [2,3];
arr.unshift(0,1);
arr.shift();
unshift后→[0,1,2,3] ; shift后→[1,2,3] 弹出0
splice()
万能修改
删除/插入/替换任意位置，直接修改原数组。
let arr = [1,2,3,4];
arr.splice(1,2,'x','y');
结果 arr: [1,'x','y',4]
delete
delete 操作符
移除元素但保留空位，长度不变（不推荐）。
let arr = [1,2,3];
delete arr[1];
[1, 空, 3] (长度仍为3)
方法 遍历 & 迭代 (高阶函数)
forEach()
简单遍历
对每个元素执行回调，无返回值。
[1,2,3].forEach(v => console.log(v*2));
控制台打印: 2, 4, 6
map()
映射新数组
每个元素经过回调，返回新数组。
let doubled = [1,2,3].map(x => x*2);
doubled = [2,4,6]
filter()
过滤元素
返回满足条件的新数组。
let evens = [1,2,3,4].filter(x => x%2===0);
evens = [2,4]
reduce()
累积归并
逐次迭代，累积为单个值。
let sum = [1,2,3].reduce((acc, v) => acc+v, 0);
sum = 6
方法 查找 & 判断
indexOf() / lastIndexOf()
查找索引
返回首次/末次索引，-1表示未找到。
[1,2,3,2].indexOf(2);
[1,2,3,2].lastIndexOf(2);
1 ; 3
includes()
是否包含
返回布尔值，能识别NaN。
[1,2,NaN].includes(NaN);
true
find() / findIndex()
条件查找
返回第一个满足条件的元素/索引。
[5,12,8].find(x => x>10);
[5,12,8].findIndex(x=>x>10);
12 ; 1
some() / every()
存在性/全性判断
some 至少一个满足；every 全部满足。
[2,4,6].some(x=>x>5);
[2,4,6].every(x=>x%2===0);
true ; true
方法 合并 & 切片 & 扁平

concat() 合并

连接多个数组/值，返回新数组。

[1,2].concat([3,4], 5);
[1,2,3,4,5]

slice() 切片

返回数组的一部分，原数组不变。支持负索引。

[1,2,3,4].slice(1,3);
[2,3]

flat() / flatMap() 扁平

拉平嵌套数组；flatMap 先 map 再 flat(1)。

[[1],[2,3]].flat();
[1,2,3]
方法 排序 & 反转
sort()
数组排序
默认按字符串排序，可传入比较函数。
[3,1,10].sort();
[3,1,10].sort((a,b)=>a-b);
默认: [1,10,3] ; 比较函数: [1,3,10]
reverse()
反转数组
颠倒元素顺序，直接修改原数组。
[1,2,3].reverse();
[3,2,1]
toSorted() / toReversed()
ES2023 新方法
返回新数组，不修改原数组。
let orig = [3,1,2];
let sorted = orig.toSorted();
sorted = [1,2,3] ; orig 不变
实用 转换 & 填充 & 静态方法
join()
数组转字符串
用分隔符连接元素，默认逗号。
['a','b','c'].join('-');
"a-b-c"
toString()
转字符串
效果等同于 join(',')。
[1,2,3].toString();
"1,2,3"
fill()
填充数组
用固定值填充指定范围，修改原数组。
[1,2,3].fill(0,1,2);
[1,0,3]
对比 修改原数组 vs 返回新数组

🔧 修改原数组 破坏性

  • push() / pop()
  • unshift() / shift()
  • splice()
  • sort() / reverse()
  • fill()
  • copyWithin()

✨ 返回新数组 非破坏性

  • map() / filter()
  • concat() / slice()
  • flat() / flatMap()
  • toSorted() / toReversed()
  • reduce() (返回累积值)

⚡ 扩展运算符 (ES6)

优雅复制/合并数组，返回新数组。

let a = [1,2];
let b = [...a, 3,4];
b = [1,2,3,4]

🧪 综合示例：数组去重 (多种方法)

const arr = [1,2,2,3,4,4,5];

// 方法1: Set + 扩展符
const unique1 = [...new Set(arr)];

// 方法2: filter + indexOf
const unique2 = arr.filter((v,i,a) => a.indexOf(v) === i);

// 方法3: reduce
const unique3 = arr.reduce((acc, v) => acc.includes(v) ? acc : [...acc, v], []);

输出: 三种方法均得到 [1,2,3,4,5]
