⚡ 事件处理 · onclick vs addEventListener

onclick 覆盖 / addEventListener叠加
// onclick 赋两次会覆盖
btn.onclick = fn1;
btn.onclick = fn2; // 只有fn2生效

// addEventListener 不覆盖
btn.addEventListener('click', fn1);
btn.addEventListener('click', fn2); // 都执行
📋 点击日志
