📋 循环语句示例

for 循环
let str = "";
for (let i = 0; i < 5; i++) {
    str += i + " ";
}
执行结果：0 1 2 3 4 说明：标准 for 循环，从 0 迭代到 4，每次拼接当前值。
while 循环
let j = 0;
let res = "";
while (j < 3) {
    res += j + " ";
    j++;
}
执行结果：0 1 2 说明：先判断条件，满足则执行循环体。
do...while 循环
let k = 0;
let doRes = "";
do {
    doRes += k + " ";
    k++;
} while (k < 3);

// 至少执行一次示例
let m = 5;
let doRes2 = "";
do {
    doRes2 += m + " ";
    m++;
} while (m < 3);
第一次结果：0 1 2 第二次结果（m=5）：5 说明：do...while 至少执行一次循环体。
for...of
const colors = ["红", "绿", "蓝"];
let out = "";
for (let color of colors) {
    out += color + " ";
}
执行结果：红 绿 蓝 说明：遍历数组元素，直接获取值。
for...in
const person = { name: "Alice", age: 30, city: "北京" };
let keys = "", values = "";
for (let key in person) {
    keys += key + " ";
    values += person[key] + " ";
}
属性名：name age city 属性值：Alice 30 北京 说明：遍历对象的可枚举属性（包括原型链）。
break 语句
// break 示例：找到第一个偶数即停止
let breakRes = "";
for (let i = 1; i <= 10; i++) {
    if (i % 2 === 0) {
        breakRes += "找到偶数 " + i;
        break;
    }
    breakRes += i + " ";
}

// 带标签的 break：跳出多层循环
let outerBreak = "";
outer: for (let i = 0; i < 3; i++) {
    for (let j = 0; j < 3; j++) {
        if (i === 1 && j === 1) break outer;
        outerBreak += `(${i},${j}) `;
    }
}
break 结果：1 找到偶数 2 带标签 break 结果：(0,0) (0,1) (0,2) (1,0) 说明：break 终止当前循环；带标签的 break 跳出指定多层循环。
continue 语句
// continue 示例：跳过偶数，只输出奇数
let continueRes = "";
for (let i = 1; i <= 5; i++) {
    if (i % 2 === 0) continue;
    continueRes += i + " ";
}
continue 结果：1 3 5 说明：continue 跳过本次迭代，继续下一次循环。

🔗 路径策略：绝对路径 vs 相对路径

相对路径的隐患
<!-- header.html 中的图片 --> <img src="picture/logo.png" alt="医院logo">

场景： header.html 被加载到不同层级的页面

  • 📄 首页 (/)：图片正常 ✅
  • 📁 /about/brief.html：浏览器请求 /about/picture/logo.png ❌ 404
相对路径基于当前页面 URL 解析，导致资源丢失。
绝对路径的可靠性
<!-- header.html 中使用绝对路径 --> <img src="/picture/logo.png" alt="医院logo">

无论页面在哪个目录：

  • 首页 (/)：请求 /picture/logo.png ✅
  • /about/brief.html：同样请求 /picture/logo.png ✅
以 / 开头，从网站根目录定位，所有页面通用。
模拟动态加载
// 模拟从 header.html 插入内容 function loadHeader() { const html = ` <div> <img src="/picture/logo.png" alt="绝对路径" width="30"> <img src="picture/logo.png" alt="相对路径" width="30"> </div> `; container.innerHTML = html; }

点击按钮后，上方会插入两个图片占位：左侧绝对路径（通常可显示），右侧相对路径可能失效（显示裂图）。

📂 文件组织与命名规范

推荐项目结构
my-hospital/
  ├── css/
  │   └── style.css
  ├── js/
  │   └── main.js
  ├── picture/
  │   ├── logo.png
  │   └── icon.svg
  ├── components/
  │   ├── header.html
  │   └── footer.html
  ├── index.html
  └── about.html
命名规范

✅ 推荐：

  • 全部小写
  • 连字符分隔：main-nav.css
  • 无空格、中文：hospital-logo.png

❌ 避免：

  • 大小写混用（Windows 不敏感，但 Linux 敏感）
  • 空格、特殊字符：image copy.png
  • 中文文件名（可能编码问题）
大小写敏感
// Linux 服务器上 /picture/logo.png ✅ /Picture/logo.png ❌ 404

线上服务器通常运行 Linux，路径区分大小写！
建议统一使用 小写字母 命名文件和文件夹。

即使本地开发不区分，部署到服务器也可能产生404。
