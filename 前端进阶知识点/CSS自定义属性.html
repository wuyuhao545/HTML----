CSS 自定义属性 var( ) 语法
⚡ 提问原样 background-color: var(--clr-accent);
color: var(--clr-light);
变量直接使用
在根作用域定义 --clr-accent 和 --clr-light，然后任何地方都能调用。
🌟 背景 = var(--clr-accent)
文字 = var(--clr-light)
:root {
  --clr-accent: #3b82f6;
  --clr-light: #f8fafc;
}
.demo {
  background: var(--clr-accent);
  color: var(--clr-light);
}
🎨 主题切换基础 --clr-accent 可动态改变
JavaScript 更新变量
修改变量值即可全局更新配色，无需逐一覆盖。
// 改变根变量
document.documentElement.style
  .setProperty('--clr-accent', '#8b5cf6');
📱 响应式中的变量 @media 内重定义
适配深色/浅色
媒体查询或数据集切换变量值，实现主题统一。
@media (prefers-color-scheme: dark) {
  :root { --clr-accent: #bb86fc; }
}

继承与后备
卡片背景使用了 rgba(255,255,255,0.85)，但也可以替换为变量，更灵活。
此处背景 var(--clr-card-bg, #ffffff)，后备为白色。
