基础 DOM 简介 & 节点类型
DOM 树
什么是 DOM？
文档对象模型将 HTML 文档表示为节点树。每个元素、属性、文本都是节点。
// 简单示例
<div id="app">Hello</div>
// 对应的节点: Element(div), Attr(id), Text("Hello")
根节点: document ; 元素节点: <div> ; 文本节点: "Hello"
nodeType
节点类型常量
每个节点有 nodeType 属性标识类型：1(元素), 2(属性), 3(文本), 8(注释), 9(文档)。
document.nodeType; // 9
document.body.nodeType; // 1
9 ; 1
nodeName / nodeValue
节点名称与值
元素节点 nodeName 为标签名，文本节点 nodeValue 为文本内容。
document.body.nodeName; // "BODY"
document.body.firstChild?.nodeValue; // 可能为null或文本
"BODY" ; 文本内容
instanceof 检查
判断节点类型
使用 instanceof 检查特定接口：Element, HTMLElement, HTMLDivElement 等。
document.body instanceof HTMLElement; // true
true
方法 选择 / 获取元素
getElementById()
通过 ID 获取
返回匹配ID的第一个元素，效率高。
document.getElementById('app');
返回对应元素或 null
getElementsByClassName()
通过类名获取
返回实时 HTMLCollection，可通过索引访问。
document.getElementsByClassName('item');
HTMLCollection(2) [div.item, div.item]
getElementsByTagName()
通过标签名获取
返回实时 HTMLCollection。可以通过索引访问特定元素，也可遍历所有元素。
// 假设页面有两个 <div> 元素
const divs = document.getElementsByTagName('div');

// 获取第二个 <div> (索引从0开始)
const secondDiv = divs[1];
console.log(secondDiv.textContent); // 输出第二个div的内容

// 遍历所有 <div>
for (let div of divs) {
    console.log(div.textContent);
}

// 或者转为数组后使用 forEach
Array.from(divs).forEach(div => console.log(div.id));
假设页面有两个div，内容分别为 "第一个" 和 "第二个"，则 divs[1] 返回第二个div，内容 "第二个"；遍历依次输出 "第一个" 和 "第二个"。
querySelector() / querySelectorAll()
CSS 选择器
querySelector 返回第一个匹配，querySelectorAll 返回静态 NodeList。
document.querySelector('.item:first-child');
document.querySelectorAll('div > p');
Element ; NodeList(2) [p, p]
属性 节点遍历 (父子兄弟)
parentNode / parentElement
父节点
parentNode 可能返回 Document 节点，parentElement 返回元素父节点。
document.body.parentNode; // <html>
document.body.parentElement; // <html>
<html>元素
children / childNodes
子节点
children 仅元素子节点，childNodes 包含文本、注释等。
document.body.children;
document.body.childNodes;
HTMLCollection ; NodeList (含换行文本等)
firstChild / lastChild
首尾子节点
包括文本节点。对应元素版本: firstElementChild / lastElementChild。
document.body.firstChild;
document.body.firstElementChild;
可能是文本节点 ; 第一个元素
nextSibling / previousSibling
兄弟节点
包括文本节点。元素版本: nextElementSibling / previousElementSibling。
document.body.nextSibling; // 通常是换行文本
文本节点或 null
方法 节点操作 (创建/插入/移除)
createElement() / createTextNode()
创建节点
创建元素节点和文本节点。
const div = document.createElement('div');
const text = document.createTextNode('Hello');
div 元素 ; 文本节点
appendChild() / append()
添加子节点
appendChild 追加一个节点，返回该节点。append 可追加多个节点/字符串，无返回值。
parent.appendChild(newChild);
parent.append('text', anotherChild);
parent 最后增加子节点
insertBefore() / insertAdjacentElement()
插入到指定位置
insertBefore( newNode, referenceNode )；insertAdjacentElement 支持 'beforebegin' 等。
parent.insertBefore(newItem, existingItem);
将 newItem 插入到 existingItem 之前
removeChild() / remove()
移除节点
removeChild 需通过父节点移除，返回被移除节点。remove 是被元素自身调用（IE 不支持）。
parent.removeChild(child);
child.remove(); // 现代浏览器
从 DOM 中移除 child
replaceChild()
替换子节点
parent.replaceChild(newChild, oldChild)。
parent.replaceChild(newSpan, oldP);
用 newSpan 替换 oldP
cloneNode()
克隆节点
参数 true 深克隆（包含子节点），false 浅克隆。
const copy = node.cloneNode(true);
返回克隆的新节点
方法 属性操作
getAttribute() / setAttribute()
标准属性方法
获取、设置任意属性（包括自定义 data-*）。
el.getAttribute('data-id');
el.setAttribute('class', 'active');
'123' ; 设置 class
hasAttribute() / removeAttribute()
判断/移除属性
检查是否存在某属性，移除属性。
el.hasAttribute('disabled');
el.removeAttribute('disabled');
true ; 移除后 disabled 消失
id / className / classList
直接属性访问
id 和 className 直接读写；classList 提供 add/remove/toggle/contains 方法。
el.id = 'main';
el.className = 'box';
el.classList.add('highlight');
设置 ID、类，添加 highlight 类
dataset
自定义数据属性
访问 data-* 属性的对象，例如 data-user 对应 dataset.user。
el.dataset.user = '张三';
设置 data-user="张三"
方法 样式操作
style 属性
内联样式
通过 element.style 属性读写内联样式（驼峰命名）。
el.style.backgroundColor = 'red';
el.style.fontSize = '16px';
内联样式添加 background-color: red;
getComputedStyle()
计算后样式
获取元素最终应用的所有 CSS 属性值（只读）。
getComputedStyle(el).color;
"rgb(0, 0, 0)"
classList 操作
通过类切换样式
add, remove, toggle, contains 管理类名。
el.classList.toggle('visible');
如果存在则移除，否则添加
方法 事件监听
addEventListener()
添加事件
绑定事件处理函数，第三个参数可选（捕获、once等）。
btn.addEventListener('click', (e) => {
    console.log('clicked');
});
点击按钮时打印 "clicked"
removeEventListener()
移除事件
需传入相同函数引用才能移除。
btn.removeEventListener('click', handler);
移除后不再触发
事件对象 e
常用属性和方法
target, currentTarget, preventDefault(), stopPropagation()。
e.preventDefault(); // 阻止默认行为
e.stopPropagation(); // 阻止冒泡
例如阻止链接跳转
事件委托
利用冒泡
在父元素监听事件，通过 target 判断子元素，适用于动态元素。
list.addEventListener('click', (e) => {
    if (e.target.matches('li')) {...}
});
点击 li 时触发
陷阱 常见坑 & 对比
HTMLCollection vs NodeList
实时与静态
getElementsBy系列返回 实时集合 (DOM 变化自动更新)；querySelectorAll 返回 静态 NodeList (快照)。
const items = document.getElementsByClassName('item');
const itemsStatic = document.querySelectorAll('.item');
// 删除一个 .item 后，items 长度减少，itemsStatic 不变
items 实时变化；itemsStatic 保持旧长度
childNodes vs children
节点与元素
childNodes 包含文本、注释，children 只包含元素节点。
document.body.childNodes.length; // 可能包含换行文本
document.body.children.length;   // 只计元素
例如 body 直接子元素个数
innerHTML vs textContent
内容操作
innerHTML 解析 HTML 标签，textContent 纯文本，更安全。
el.innerHTML = '<b>Hi</b>';  // 加粗
el.textContent = '<b>Hi</b>'; // 显示原文
第一个显示加粗 Hi，第二个显示 <b>Hi</b>
offsetWidth / clientWidth
尺寸区别
offsetWidth 包括边框、滚动条；clientWidth 包括内边距，不含边框。
el.offsetWidth; // 完整外宽
el.clientWidth; // 可视内容宽
举例：边框+内边距+宽度

🧪 综合示例：动态待办列表 (DOM 操作 + 事件委托)

<ul id="todo-list">
    <li>学习 DOM</li>
    <li>写笔记</li>
</ul>
<input id="newTodo" placeholder="新任务">
<button id="addBtn">添加</button>

<script>
const list = document.getElementById('todo-list');
const input = document.getElementById('newTodo');
document.getElementById('addBtn').addEventListener('click', () => {
    const text = input.value.trim();
    if (text) {
        const li = document.createElement('li');
        li.textContent = text;
        list.appendChild(li);
        input.value = '';
    }
});
// 事件委托：点击 li 标记完成
list.addEventListener('click', (e) => {
    if (e.target.tagName === 'LI') {
        e.target.classList.toggle('done');
    }
});
</script>

说明: 点击按钮添加任务，点击任务项切换完成状态 (需要预先定义 .done 样式)
