🧩 JavaScript Element 对象·完全笔记

继承自 Node · 元素专属属性与方法大全

基础 Element 是什么？
Element 接口
文档中元素的基础对象
Element 继承自 Node，是所有 HTML 或 XML 元素对象的基类。它提供了元素特有的属性和方法，如 id、className、classList、innerHTML 以及尺寸、滚动等操作。 
// 任何元素都是 Element 的实例
document.body instanceof Element;  // true
document.body instanceof Node;      // true
true ; true
属性 标识与内容
id / className / tagName
基本标识
id 元素唯一标识，className 类名字符串，tagName 标签名（大写）。
el.id = 'main';
el.className = 'box red';
console.log(el.tagName); // 'DIV'
设置 id 和 class，输出标签名 'DIV'
classList
类名操作
提供 add/remove/toggle/contains 方法管理类名。
el.classList.add('active');
el.classList.toggle('visible');
el.classList.contains('box'); // true
添加类、切换类、检查类是否存在
innerHTML / outerHTML
HTML 内容
innerHTML 设置或获取内部的HTML；outerHTML 包含自身标签。
el.innerHTML = '<span>Hi</span>';
console.log(el.outerHTML); // '<div><span>Hi</span></div>'
innerHTML 修改内部，outerHTML 输出完整标签
textContent
纯文本内容
获取或设置元素的文本（不解析 HTML）。
el.textContent = 'Hello <b>World</b>'; // 不会加粗
元素内显示 "Hello <b>World</b>" 纯文本
遍历 父子兄弟 & 查找
children / parentElement
父子元素
children 返回子元素集合（HTMLCollection）；parentElement 返回父元素。
el.children;      // HTMLCollection
el.parentElement; // 父元素或 null
获取所有子元素和父元素
firstElementChild / lastElementChild
首尾子元素
只返回元素节点，忽略文本/注释。
el.firstElementChild; // 第一个子元素
el.lastElementChild;  // 最后一个子元素
与 firstChild 不同，它不会返回文本节点
nextElementSibling / previousElementSibling
兄弟元素
返回下一个/上一个兄弟元素，忽略非元素节点。
el.nextElementSibling;
el.previousElementSibling;
例如两个 <div> 之间若有换行，依然能正确获取相邻元素
closest(selector) / matches(selector)
选择器匹配
closest 返回匹配的最近祖先（包括自身）；matches 检查自身是否匹配。
el.closest('.container');
el.matches('.active');   // true/false
找到最近的 .container 祖先；检查自身是否有 active 类
方法 属性操作
getAttribute() / setAttribute()
获取/设置属性
操作任何属性（包括自定义 data-*）。
el.setAttribute('data-id', '123');
console.log(el.getAttribute('data-id')); // '123'
设置自定义属性并获取
hasAttribute() / removeAttribute()
判断/移除属性
检查是否存在属性，移除属性。
el.hasAttribute('disabled'); // true/false
el.removeAttribute('disabled');
移除 disabled 属性后按钮可用
dataset
自定义数据
通过 dataset 对象读写 data-* 属性。
el.dataset.user = '张三';
console.log(el.dataset.user); // '张三'
对应 HTML 中 data-user="张三"
样式 内联与计算
style 属性
内联样式
读写元素的内联样式（驼峰命名）。
el.style.backgroundColor = 'red';
el.style.fontSize = '16px';
添加 background-color: red; 到内联样式
getComputedStyle()
计算后样式
获取元素最终应用的所有 CSS 属性值（只读）。
getComputedStyle(el).color; // 'rgb(0, 0, 0)'
获取当前实际生效的颜色值
尺寸 宽高与位置
clientWidth / clientHeight
可视内容宽高
包含内边距，不含边框和滚动条。
el.clientWidth;  // 例如 200
元素内容区宽度（padding + width）
offsetWidth / offsetHeight
布局宽高
包含边框、内边距和滚动条（如果有）。
el.offsetWidth; // 例如 220 (包含边框)
元素占用的完整宽高
scrollWidth / scrollHeight
滚动尺寸
包含溢出内容的完整尺寸。
el.scrollWidth;  // 内容总宽度
当内容溢出时可获取完整宽高
getBoundingClientRect()
相对于视口的位置
返回 DOMRect 对象，包含 top, right, bottom, left, width, height。
const rect = el.getBoundingClientRect();
console.log(rect.top, rect.left);
元素左上角距离视口顶部/左边的距离
行为 滚动与插入
scrollIntoView()
滚动到可视区
使元素滚动到可见区域，可配置滚动行为和对齐方式。
el.scrollIntoView({ behavior: 'smooth', block: 'center' });
页面平滑滚动，元素位于视口中央
insertAdjacentHTML()
插入 HTML
在指定位置插入 HTML 字符串，位置：'beforebegin', 'afterbegin', 'beforeend', 'afterend'。
el.insertAdjacentHTML('beforeend', '<span>Hi</span>');
在元素内部末尾添加 <span>Hi</span>
insertAdjacentElement() / insertAdjacentText()
插入元素或文本
类似 insertAdjacentHTML，但参数分别为元素节点和文本字符串。
const span = document.createElement('span');
el.insertAdjacentElement('beforeend', span);
el.insertAdjacentText('afterbegin', 'Hello');
在元素内部开头插入文本，末尾插入 span 元素
操作 添加/删除/替换 (Node 方法)
append() / prepend()
追加/前置子节点
可同时添加多个节点或字符串，无返回值。
parent.append('文本', childElement);
parent.prepend(childElement);
在父元素末尾/开头插入内容
before() / after()
在元素前后插入
将节点或字符串插入到当前元素之前或之后。
el.before('前面', someElement);
el.after('后面', anotherElement);
在当前元素前后添加内容
replaceWith()
替换自身
用指定的节点或字符串替换当前元素。
el.replaceWith(newElement, '一些文本');
el 被 newElement 和文本节点替换
remove()
移除自身
直接从 DOM 中删除该元素。
el.remove(); // 元素消失
从文档中移除 el

🧪 综合示例：Element 方法小实践

<div id="box" class="card" data-info="demo">原始内容</div>

<script>
const box = document.getElementById('box');

// 1. 属性操作
box.setAttribute('data-size', 'large');
console.log(box.dataset.size);   // 'large'

// 2. 样式修改
box.style.backgroundColor = '#ffd966';
box.classList.add('highlight');

// 3. 插入新元素
const btn = document.createElement('button');
btn.textContent = '点击';
box.after(btn);

// 4. 尺寸信息
console.log(box.getBoundingClientRect().top);

// 5. 查找祖先
const container = box.closest('.container');
</script>

说明: 演示了属性、样式、插入、尺寸和查找等 Element 常用方法。
