📌 JavaScript 事件类型大全

鼠标 · 键盘 · 表单 · 焦点 · 滚动 · 触摸 · 拖拽

🖱️ 鼠标事件 点击、移动、滚轮
click / dblclick
单击和双击事件。
element.addEventListener('click', (e) => {
    console.log('单击');
});
element.addEventListener('dblclick', (e) => {
    console.log('双击');
});
单击触发一次，双击触发两次单击和一次双击。
mousedown / mouseup
鼠标按下和释放。
element.addEventListener('mousedown', (e) => {
    console.log('按下', e.button);
});
element.addEventListener('mouseup', (e) => {
    console.log('释放');
});
可配合判断按键 (0左,1中,2右)。
mousemove
鼠标在元素内移动时持续触发。
element.addEventListener('mousemove', (e) => {
    console.log(`坐标: ${e.clientX}, ${e.clientY}`);
});
常用于拖拽或坐标跟踪。
mouseenter / mouseleave
鼠标进入/离开元素（不冒泡）。
element.addEventListener('mouseenter', () => {
    console.log('进入');
});
element.addEventListener('mouseleave', () => {
    console.log('离开');
});
进入子元素不会触发父元素的 leave/enter。
mouseover / mouseout
鼠标进入/离开元素（冒泡）。
element.addEventListener('mouseover', () => {
    console.log('over');
});
element.addEventListener('mouseout', () => {
    console.log('out');
});
移动到子元素上也会触发父元素的 over/out。
wheel
鼠标滚轮滚动。
element.addEventListener('wheel', (e) => {
    console.log('滚动方向', e.deltaY > 0 ? '下' : '上');
    e.preventDefault(); // 阻止页面滚动
});
可获取滚动增量及方向。
⌨️ 键盘事件 按键按下、释放
keydown / keyup
任意键按下和释放。
document.addEventListener('keydown', (e) => {
    console.log(`按下: ${e.key} (code: ${e.code})`);
});
document.addEventListener('keyup', (e) => {
    console.log('释放');
});
e.key 返回字符，e.code 返回物理按键。
keypress (已弃用)
字符键按下时触发（不推荐，建议用 keydown）。
element.addEventListener('keypress', (e) => {
    console.log(String.fromCharCode(e.charCode));
});
仅响应可打印字符，现代开发改用 keydown。
修饰键判断
检测 Ctrl、Shift、Alt 等。
document.addEventListener('keydown', (e) => {
    if (e.ctrlKey && e.key === 's') {
        e.preventDefault();
        console.log('保存操作');
    }
});
通过 e.ctrlKey、e.shiftKey、e.altKey 判断。
📋 表单事件 输入、提交、重置等
input / change
input 实时触发，change 在失去焦点且值变化时触发。
input.addEventListener('input', (e) => {
    console.log('当前值:', e.target.value);
});
input.addEventListener('change', (e) => {
    console.log('最终值:', e.target.value);
});
适用于文本输入、复选框等。
submit
表单提交时触发，可阻止默认提交。
form.addEventListener('submit', (e) => {
    e.preventDefault();
    console.log('表单被提交');
});
常用于前端验证或 AJAX 提交。
reset
表单重置时触发。
form.addEventListener('reset', () => {
    console.log('表单重置');
});
点击重置按钮时触发。
focus / blur
元素获得/失去焦点。
input.addEventListener('focus', () => {
    console.log('获得焦点');
});
input.addEventListener('blur', () => {
    console.log('失去焦点');
});
可用于输入框高亮或验证。
📜 其他事件 滚动、触摸、拖拽
scroll
元素或窗口滚动时触发。
window.addEventListener('scroll', () => {
    console.log('滚动位置:', window.scrollY);
});
注意节流，避免高频触发影响性能。
touchstart / touchmove / touchend
触摸事件，用于移动端。
element.addEventListener('touchstart', (e) => {
    console.log('触摸点:', e.touches);
});
element.addEventListener('touchmove', (e) => {
    e.preventDefault(); // 阻止滚动
});
可获取多个触摸点，实现手势。
dragstart / drag / dragend
HTML5 拖拽事件。
draggableElement.addEventListener('dragstart', (e) => {
    e.dataTransfer.setData('text/plain', 'some data');
});
需设置元素 draggable="true"。
resize
窗口大小改变时触发。
window.addEventListener('resize', () => {
    console.log('新尺寸:', window.innerWidth, window.innerHeight);
});
常用于响应式调整。
⚡ 事件类型是交互的基石 · 根据需求选择恰当的事件
