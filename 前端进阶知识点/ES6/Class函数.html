📌 基础类 · 构造函数与方法

class 声明与构造函数实例化
class Animal {
  constructor(name) {
    this.name = name;
  }
  speak() {
    return `${this.name} 发出声音`;
  }
}
const dog = new Animal("旺财");
🐕 通过 constructor 初始化实例属性，方法自动添加至原型。
类表达式匿名/命名
const Person = class {
  constructor(name) { this.name = name; }
  greet() { return `Hi, I'm ${this.name}`; }
};
const me = new Person("Alice");
📦 类也可以作为表达式使用，支持立即执行。
Getter / Setter属性拦截
class Circle {
  constructor(radius) { this._radius = radius; }
  get area() { return Math.PI * this._radius ** 2; }
  set radius(r) {
    if (r > 0) this._radius = r;
  }
}
📏 通过 getter/setter 实现属性访问控制与计算属性。

🧬 继承 · super · 静态成员

extends 继承super调用父类
class Dog extends Animal {
  constructor(name, breed) {
    super(name);
    this.breed = breed;
  }
  speak() {
    return `${super.speak()} 汪汪！`;
  }
}
🐕‍🦺 子类必须调用 super，可重写父类方法。
静态方法 & 静态属性类级别
class MathUtils {
  static PI = 3.14159;
  static square(x) { return x * x; }
}
console.log(MathUtils.PI); // 3.14159
console.log(MathUtils.square(5)); // 25
🔧 静态成员属于类本身，无需实例化，常用于工具方法。
静态继承中的 this多态表现
class Parent {
  static name = "Parent";
  static who() { return this.name; }
}
class Child extends Parent {
  static name = "Child";
}
console.log(Parent.who()); // "Parent"
console.log(Child.who()); // "Child"
🎯 静态方法中的 this 指向调用它的类（动态绑定），实现静态多态。
instanceof 检查原型链判断
const dog = new Dog("旺财", "金毛");
console.log(dog instanceof Dog); // true
console.log(dog instanceof Animal); // true
🔍 判断对象是否属于某个类或其父类。

🔐 私有字段 & 高级特性

私有字段 (#)ES2022
class Wallet {
  #balance = 0;
  add(amount) { this.#balance += amount; }
  getBalance() { return this.#balance; }
}
const myWallet = new Wallet();
myWallet.add(100);
🔐 # 开头的字段为私有，外部无法直接访问，真正封装。
私有方法内部使用
class Counter {
  #count = 0;
  #validate() { return true; }
  increment() {
    if (this.#validate()) this.#count++;
  }
  get value() { return this.#count; }
}
📦 私有方法同样通过 # 前缀定义，外部不可调用。
类静态块ES2022
class Database {
  static config;
  static {
    this.config = { host: 'localhost' };
  }
}
⚙️ 静态块用于初始化静态属性，执行复杂逻辑。
