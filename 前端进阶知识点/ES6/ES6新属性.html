🔒 块级王者 · let / const 与暂时性死区

let · 块级作用域隔离 无变量提升污染
let message = "全局/外部";
if (true) {
let message = "内部块级";
console.log("内部:", message);
}
console.log("外部:", message);
✅ let 声明的变量只在当前代码块内生效，避免全局污染，提升代码健壮性。
const · 只读引用 对象属性可变
const PI = 3.14159;
// PI = 3.14; ❌ 报错
const user = { name: "Alex" };
user.name = "Jordan"; // ✅ 对象属性可修改
🔒 const 保证变量指向的地址不变，但对象/数组内部属性可以自由修改。
暂时性死区 (TDZ) · 安全机制
// console.log(value); // ❌ ReferenceError
let value = "ES6 安全宣言";
console.log("访问成功:", value);
⚡ 在声明之前访问 let/const 变量会触发 ReferenceError，避免旧的提升陷阱。

🎁 对象解构赋值 · 重命名 & 嵌套萃取 & 参数对象解构

对象解构 & 重命名 & 默认值
const employee = {
id: 1024,
fullName: "欧阳静",
department: "技术研发"
};
const { fullName: userName, department, level = "P5" } = employee;
console.log(userName, department, level);
🏷️ 提取属性同时重命名，轻松设置默认值，代码清晰优雅。
嵌套解构 & 函数参数解构 深度提取
const apiResponse = {
status: 200,
data: {
user: { id: 556, nickname: "深潜者", posts: [12,33] }
}
};
const { data: { user: { id: userId, nickname } } } = apiResponse;
function introduce({ name, age = 24, city = "未知" }) {
return `${name} · ${age}岁 · 来自${city}`;
}
🧩 深层数据一次性解构，函数参数直接解析对象，减少样板代码。
