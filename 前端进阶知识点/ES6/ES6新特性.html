⚡ ES6+ 核心特性实战

现代 JavaScript · 更简洁 · 更强大 · 更优雅

📌 涵盖 ES2015(ES6) 至 ES2022 常用新特性，每个示例均附带运行效果

🔹 块级作用域 · let & const

let 变量 · 块级隔离 无提升陷阱
let score = 100;
if (true) {
let score = 200;
console.log("内部:", score);
}
console.log("外部:", score);
✅ let 声明的变量只在当前代码块内生效，避免变量污染。
const 常量 · 不可重新赋值
const PI = 3.14159;
// PI = 3.14; ❌ 报错
const user = { name: "Alice" };
user.name = "Bob"; // ✅ 对象属性可变
🔒 const 保证绑定不可变，但对象/数组内部属性可以修改。
暂时性死区 (TDZ) 演示
// console.log(a); // 报错 TDZ
let a = 10;
console.log("a =", a);
⚡ 在声明之前访问 let/const 变量会触发 ReferenceError，提升安全性。

➡️ 箭头函数 · 简洁 & this 穿透

极简语法 & 隐式返回
const double = n => n * 2;
const sum = (a, b) => a + b;
const greet = () => "Hello ES6!";
🚀 单参数可省略括号，单表达式自动返回，代码更精简。
箭头函数中的 this 词法绑定
const obj = {
name: "箭头世界",
show: function() {
setTimeout(() => {
console.log(this.name);
}, 10);
}
};
obj.show();
🎯 箭头函数没有自己的 this，继承外层作用域的 this 值，解决回调 this 丢失问题。
返回对象字面量 小技巧
const makeUser = (name,age) => ({ name, age });
const user = makeUser("Eva", 28);
📦 需要用括号包裹对象字面量，避免与函数体歧义。

📦 解构赋值 · 数组 & 对象 & 嵌套

数组解构 + 默认值
const [first, second = 100] = [10];
const [x, y, ...rest] = [1, 2, 3, 4, 5];
✨ 交换变量、剩余模式，快速提取数组元素。
对象解构 & 重命名
const person = { id: 7, fullName: "李华", city: "上海" };
const { fullName: name, city } = person;
🏷️ 提取属性同时重命名，设置默认值更灵活。
嵌套解构 & 函数参数解构
const response = { data: { user: { id: 101, posts: [1,2] } } };
const { data: { user: { id } } } = response;
function greet({ name, age = 20 }) { return `${name} ${age}岁`; }
🧩 深层数据提取，函数参数直接解构对象，代码清晰。

📝 模板字符串 · 多行 & 插值 & 标签函数

插值与表达式
const user = "王芳", points = 95;
`欢迎 ${user}, 您的积分: ${points + 5}`
💡 支持任意 JavaScript 表达式，自动转字符串拼接。
多行字符串 · 保留格式
const poem = `\n 明月出天山，\n 苍茫云海间。\n 长风几万里，\n 吹度玉门关。`;
📜 无需 \n 拼接，保留缩进和换行，优雅呈现文本。
标签模板 (Tag Function)
function highlight(strings, ...values) {
return strings.reduce((acc,str,i) => `${acc}${str}${values[i]||''}`,'');
}
const price = 299;
highlight`总价: ${price} 元`;
🎨 自定义模板解析，用于过滤、国际化或样式高亮。

🔁 扩展运算符(...) & 剩余参数

数组展开 & 合并
const arr1 = [1,2,3];
const arr2 = [4,5,6];
const combined = [...arr1, ...arr2];
const copy = [...arr1];
🧬 浅拷贝、合并数组、函数传参极简写法。
对象扩展 (ES2018+)
const base = { a: 1, b: 2 };
const extended = { ...base, c: 3, b: 99 };
📘 合并对象，后覆盖前，创建新对象（浅拷贝）。
剩余参数 · 函数不定参
function sumAll(...numbers) {
return numbers.reduce((acc, n) => acc + n, 0);
}
sumAll(2,4,6,8);
➕ 代替 arguments，获得真正数组，灵活处理任意数量参数。

⏳ Promise 与 async/await · 优雅异步

Promise 链式调用
const fetchUser = () => Promise.resolve({ id:1, name:"Alex" });
fetchUser().then(user => user.name).then(console.log);
🤝 解决回调地狱，状态明确，支持 .catch 统一错误处理。
async/await 同步风格
async function getData() {
const res = await fetchUser();
return `用户: ${res.name}`;
}
getData().then(console.log);
🎭 基于 Promise 的语法糖，让异步代码像同步一样可读。
Promise.all 并行
const p1 = Promise.resolve(3);
const p2 = 42;
const p3 = new Promise((resolve) => setTimeout(resolve, 30, 'foo'));
Promise.all([p1,p2,p3]).then(values => console.log(values));
⚡ 多个 Promise 并发执行，等待全部完成或任一失败。

🏛️ Class 类 · 面向对象优雅语法

类声明 & 构造函数
class Animal {
constructor(name) { this.name = name; }
speak() { return `${this.name} 发出声音`; }
}
const dog = new Animal("小狗");
🐕 更清晰的构造器、方法定义，支持 extends 继承。
静态方法与继承
class Cat extends Animal {
constructor(name, color) { super(name); this.color = color; }
static info() { return "猫科动物"; }
}
🧬 super 调用父类，static 定义类级别方法，代码组织更专业。
私有字段 (ES2022) 预览
class Wallet {
#balance = 0;
add(amount) { this.#balance += amount; }
getBalance() { return this.#balance; }
}
🔐 # 开头的私有属性，真正封装内部状态，防止外部访问。

📦 模块化 · 可选链 · 新式特性

ES Module 语法 (概念)
// export const version = "1.0";
// import { version } from './module.js';
🌐 原生模块化，静态导入导出，支持 tree shaking，现代工程标准。
可选链 ?. 安全访问
const userData = null;
console.log(userData?.profile?.name ?? "匿名");
🛡️ 避免深层对象访问时报错，短路返回 undefined，结合 ?? 更佳。
空值合并 ?? 与逻辑或区别
const count = 0;
console.log(count || 100); // 100
console.log(count ?? 100); // 0
🎯 只有 null/undefined 才返回默认值，0/'' 视为有效。
