🌀 Set · 集合数据结构

唯一值存储 · 高效查找 · 迭代与集合运算

✨ ES6 新增数据结构，元素不可重复，遍历顺序即插入顺序

📌 Set 基础操作 · 增删查改

创建 Set & 添加元素 add / has
const set = new Set();
set.add(1);
set.add(2);
set.add(2); // 重复无效
set.add('hello');
console.log(set.size); // 3
📌 add 返回 Set 对象本身，支持链式调用。Set 中 NaN 相等，+0 与 -0 被视为相同。
删除 & 清空 delete / clear
set.delete(2); // 返回布尔值
set.clear(); // 清空所有元素
🗑️ delete 删除指定值，clear 清空集合。
size & has 检查 大小与存在性
const fruits = new Set(['🍎', '🍊']);
console.log(fruits.size); // 2
console.log(fruits.has('🍎')); // true
🔍 has 方法快速判断值是否存在，时间复杂度 O(1)。

🔄 Set 遍历 · 迭代方法

forEach 遍历 与数组类似
const colors = new Set(['红', '绿', '蓝']);
colors.forEach((value, key, set) => {
console.log(value); // 红 绿 蓝
});
📋 forEach 的回调参数 value 和 key 相同，保持与 Map 一致。
for...of 迭代 直接获取值
for (let item of colors) {
console.log(item);
}
🌀 Set 是可迭代对象，支持 for...of 和扩展运算符。
keys() / values() / entries() 迭代器
const it = set.values();
// keys() 同 values()，entries() 返回 [value,value]
📎 这些方法返回迭代器对象，便于配合 for...of 或手动遍历。

🔄 数组 ↔ Set 互转 & 经典去重

数组去重 · 经典用法 性能优秀
const arr = [1, 2, 2, 3, 4, 4, 5];
const unique = [...new Set(arr)];
// unique → [1,2,3,4,5]
🚀 利用 Set 唯一性快速去重，扩展运算符转回数组。
Set 转数组 & 数组转 Set
const setFromArr = new Set([1,2,3]);
const arrFromSet = Array.from(setFromArr);
// 或 [...set]
📦 灵活转换，适用于不同场景需求。
字符串去重 字符唯一化
const str = "abracadabra";
const uniqueChars = [...new Set(str)].join('');
// "abrcd"
🔤 Set 同样可用于字符串字符去重。

🧮 集合运算 · 并集 / 交集 / 差集

并集 Union A ∪ B
const a = new Set([1,2,3]);
const b = new Set([3,4,5]);
const union = new Set([...a, ...b]);
➕ 合并两个 Set 去重。
交集 Intersection A ∩ B
const intersection = new Set([...a].filter(x => b.has(x)));
🔗 筛选同时存在于两个集合的元素。
差集 Difference A \ B
const difference = new Set([...a].filter(x => !b.has(x)));
➖ 在 A 中但不在 B 中的元素。

🗑️ WeakSet · 弱引用集合

WeakSet 特性 只存储对象
const ws = new WeakSet();
let obj = {};
ws.add(obj);
// obj = null; 之后对象可被垃圾回收
🧹 成员只能是对象，弱引用不影响垃圾回收，不可遍历。
💡 Set 是高效且强大的数据结构，适用于去重、集合运算、管理唯一值等场景。
