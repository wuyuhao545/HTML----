➡️ 函数新特性 · 箭头函数 & 参数进化

箭头函数 · 极简语法 隐式返回
const double = n => n * 2;
const sum = (a, b) => a + b;
const greet = () => "Hello ES6+";
const getUser = (name, age) => ({ name, age });
🚀 单参数可省略括号，单表达式自动返回，返回对象需加括号。
箭头函数 this · 词法绑定 无自己的 this
const counter = {
count: 0,
start: function() {
setInterval(() => {
this.count++;
console.log(this.count);
}, 500);
}
};
// counter.start();
🎯 箭头函数继承外层 this，解决回调中 this 丢失问题。
参数默认值 · 简化逻辑
function greetUser(name = "访客", level = 1) {
return `欢迎 ${name}，等级 ${level}`;
}
greetUser();
greetUser("Alice", 5);
📌 为参数提供默认值，避免 undefined 判断，代码更健壮。

📦 函数高级参数 · 剩余参数 & 参数解构

剩余参数 ...args 替代 arguments
function sumAll(...numbers) {
return numbers.reduce((acc, n) => acc + n, 0);
}
sumAll(2, 4, 6, 8);
➕ 收集剩余参数为真数组，支持任意数量参数。
函数参数解构 对象 & 数组解构
function printUser({ name, age = 20, city = "未知" }) {
return `${name} · ${age}岁 · 来自${city}`;
}
printUser({ name: "林青", age: 28 });
🧩 直接在参数中解构对象/数组，设置默认值，减少样板代码。

🏛️ 对象增强 · 简写 & 计算属性 & 新方法

属性简写 & 方法简写
const name = "陈瑶";
const age = 26;
const user = { name, age };
const obj = {
sayHi() { return "Hi"; }
};
📘 变量名直接作为属性，方法省略 :function，代码更清爽。
计算属性名 · 动态 key
const key = "dynamicKey";
const obj = {
[key]: "动态值",
[`prefix_${Math.random()}`]: 42
};
✨ 使用方括号动态定义对象属性名，灵活构建对象。
Object.assign · 合并/克隆
const target = { a: 1 };
const source = { b: 2, c: 3 };
const result = Object.assign(target, source);
// result === target
🔗 浅拷贝合并对象，常用于复制或混入属性。

🔐 对象安全与比较 · 扩展运算符 & 严格相等

对象扩展运算符 (ES2018) 浅拷贝/合并
const base = { x: 1, y: 2 };
const extra = { y: 99, z: 3 };
const merged = { ...base, ...extra };
const clone = { ...base };
📦 优雅合并对象，后覆盖前，创建新对象（浅拷贝）。
Object.is · 严格相等增强
Object.is(NaN, NaN); // true
Object.is(+0, -0); // false
Object.is(5, "5"); // false
⚖️ 比 === 更精确，识别 NaN 与 ±0 差异。
可选链 ?. & 空值合并 ?? 安全访问
const userData = null;
console.log(userData?.profile?.name ?? "匿名");
const count = 0;
console.log(count ?? 100); // 0
🛡️ 短路访问深层属性，?? 仅对 null/undefined 取默认值。
