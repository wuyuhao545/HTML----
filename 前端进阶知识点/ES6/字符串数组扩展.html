📝 字符串 · 模板字面量 & 标签函数

模板字符串 · 插值与表达式 原生多行
const user = "林悦";
const score = 87;
const message = `考生 ${user} 得分: ${score + 3} 分，${score >= 80 ? "优秀" : "加油"}`;
console.log(message);
💡 反引号内支持任意 JS 表达式，自动拼接，告别繁琐字符串连接。
多行字符串 · 保留缩进/换行
const poem = `
半亩方塘一鉴开，
天光云影共徘徊。
问渠那得清如许？
为有源头活水来。`;
📜 无需 \n 拼接，保留原始格式，诗歌/模板更直观。
标签模板 · 自定义解析 高级过滤
function currency(strings, ...values) {
return strings.reduce((acc, str, i) => `${acc}${str}¥${values[i]?.toFixed(2) || ''}`, '');
}
const price = 2999.9;
const discount = 0.8;
currency`总价: ${price} 折扣后: ${price * discount}`;
🏷️ 标签函数可格式化、转义或国际化，处理模板字符串的原始内容。

🔁 数组 · 扩展运算符(...) & 剩余参数 & 解构赋值

扩展运算符 · 数组展开/合并
const arrA = [1, 2, 3];
const arrB = [4, 5, 6];
const merged = [...arrA, ...arrB];
const copy = [...arrA];
Math.max(...arrA);
🧬 浅拷贝、合并数组、函数传参极简写法，替代 apply。
剩余参数 · 收集不定参数
function sumAll(...numbers) {
return numbers.reduce((sum, n) => sum + n, 0);
}
sumAll(5, 10, 15, 20);
➕ 代替 arguments，得到真数组，灵活处理任意数量参数。
数组解构 · 快速取值 & 默认值
const colors = ["红", "绿"];
const [primary, secondary = "蓝", ...others] = colors;
console.log(primary, secondary, others);
🎨 交换变量、跳过元素、剩余模式，数据提取更灵活。

⚙️ 数组新方法 · find/findIndex/includes/flat 等

Array.prototype.find / findIndex
const users = [
{ id: 1, name: "子轩" },
{ id: 2, name: "一诺" }
];
const target = users.find(u => u.id === 2);
const idx = users.findIndex(u => u.name === "子轩");
🔍 返回第一个满足条件的元素/索引，比 filter 更精准。
Array.prototype.includes 包含判断
const fruits = ["🍎", "🍊", "🍇"];
console.log(fruits.includes("🍊")); // true
console.log(fruits.includes("🍉")); // false
✅ 直观判断数组是否包含某元素，替代 indexOf !== -1。
flat / flatMap · 扁平化 & 映射展平 ES2019
const nested = [1, [2, 3], [4, [5, 6]]];
const flatOnce = nested.flat(1);
const flatDeep = nested.flat(2);
const phrases = ["hello world", "good morning"];
const words = phrases.flatMap(p => p.split(" "));
🌀 flat 按深度展平，flatMap 映射后自动扁平化一层。

🏭 数组构造增强 · from / of & 迭代器扩展

Array.from 类数组转数组
const nodeList = document.querySelectorAll("div"); // 伪数组
const arrFromNode = Array.from(nodeList);
Array.from("ES6", char => char + "✨");
// 生成数字序列
🔁 将类数组、可迭代对象转为数组，支持 map 函数。
Array.of 避免 Array() 陷阱
const arr1 = Array.of(7); // [7]
const arr2 = Array(7); // 7个空槽位
const arr3 = Array.of(1,2,3); // [1,2,3]
📦 无论参数个数，始终创建包含所有参数的数组，行为一致。
数组空位 & 迭代扩展 概念演示
const arr = [1, , 3]; // 稀疏数组
// forEach 跳过空位，但 ES6 方法将空位视为 undefined
const newArr = Array.from(arr, x => x ?? "填充");
🧹 ES6 方法统一处理空位为 undefined，更安全。
