⏳ Async/Await · 同步风格异步编程

async/await 语法 · try/catch 错误处理 · 并发控制 · 取消请求

async/await 基本用法try/catch
async function fetchUser() {
  try {
    const res = await fetch('https://jsonplaceholder.typicode.com/users/2');
    const user = await res.json();
    console.log(user.name);
  } catch (err) { console.error(err); }
}
🎯 使用 await 等待 Promise，try/catch 捕获错误。
并发请求 · Promise.all性能优化
async function getAll() {
  const [posts, users] = await Promise.all([
    fetch('/posts/1').then(r=>r.json()),
    fetch('/users/1').then(r=>r.json())
  ]);
  console.log(posts, users);
}
⚡ 同时发起多个请求，等待全部完成。
AbortController · 取消请求中断控制
const controller = new AbortController();
fetch(url, { signal: controller.signal });
controller.abort(); // 取消请求
✋ 用于搜索框防抖、组件卸载时取消未完成的请求。
💡 async/await 是目前最推荐的异步方案，代码清晰，易于维护。
