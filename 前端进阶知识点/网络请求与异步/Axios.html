📦 Axios · 现代化HTTP客户端

Promise 风格 · 请求/响应拦截器 · 取消请求 · 并发控制

GET 请求基础用法
axios.get('https://jsonplaceholder.typicode.com/posts/1')
  .then(response => console.log(response.data))
  .catch(error => console.error(error));
✨ Axios 自动解析 JSON，响应数据在 data 字段中。
POST 请求发送数据
axios.post('https://jsonplaceholder.typicode.com/posts', {
  title: 'Axios', body: '测试', userId: 1
}).then(res => console.log(res.data));
📨 自动序列化 JSON，简化 POST 流程。
并发请求Promise.all
Promise.all([
  axios.get('/posts/1'),
  axios.get('/users/1')
]).then(([post, user]) => {
  console.log(post.data, user.data);
});
⚡ 同时发起多个请求，等待全部完成。
请求拦截器添加Token
axios.interceptors.request.use(config => {
  config.headers.Authorization = 'Bearer token';
  return config;
});
🔐 全局添加认证头、loading 状态等。
取消请求AbortController
const controller = new AbortController();
axios.get('/api', { signal: controller.signal });
controller.abort(); // 取消请求
✋ 用于搜索防抖、组件卸载时取消未完成请求。
创建实例 & 全局配置baseURL
const instance = axios.create({
  baseURL: 'https://api.example.com',
  timeout: 5000
});
instance.get('/users');
📦 针对不同服务配置独立的实例。
💡 Axios 是目前最流行的 HTTP 客户端，支持浏览器和 Node.js，推荐用于现代前端项目。
