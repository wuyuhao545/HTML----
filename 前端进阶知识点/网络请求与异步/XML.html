📡 XMLHttpRequest · 经典网络请求

GET / POST / 错误处理 / 超时控制

GET 请求获取数据
const xhr = new XMLHttpRequest();
xhr.open('GET', 'https://jsonplaceholder.typicode.com/posts/1');
xhr.onload = function() {
  if (xhr.status === 200) console.log(xhr.responseText);
};
xhr.send();
🔧 监听 onload 事件，手动解析JSON。
POST 请求发送数据
const xhr = new XMLHttpRequest();
xhr.open('POST', 'https://jsonplaceholder.typicode.com/posts');
xhr.setRequestHeader('Content-Type', 'application/json');
xhr.send(JSON.stringify({ title: 'XHR POST', body: '示例', userId: 1 }));
📤 设置请求头，发送JSON字符串。
错误处理 & 超时健壮性
xhr.timeout = 3000;
xhr.ontimeout = () => console.log('超时');
xhr.onerror = () => console.log('网络错误');
⚠️ 处理网络异常、超时和HTTP错误状态。
💡 XMLHttpRequest 是 Ajax 技术的基石，现代开发中多被 Fetch 替代，但理解其原理仍很重要。
