🤝 Promise 网络请求 · Fetch 与链式调用

Promise 风格 · 链式调用 · 错误捕获 · 并发控制

Fetch GET · 链式调用Promise
fetch('https://jsonplaceholder.typicode.com/todos/1')
  .then(res => res.json())
  .then(data => console.log(data))
  .catch(err => console.error(err));
✨ 返回Promise，自动处理响应，需调用 .json() 解析。
Fetch POST · 发送JSON配置项
fetch('https://jsonplaceholder.typicode.com/posts', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ title: 'Promise', body: '示例', userId: 1 })
}).then(res => res.json()).then(console.log);
📨 使用 method 和 body 选项发送数据。
Promise.all 并发性能优化
Promise.all([
  fetch('/posts/1').then(r=>r.json()),
  fetch('/users/1').then(r=>r.json())
]).then(([post, user]) => {
  console.log(post, user);
});
⚡ 同时发起多个请求，等待全部完成。
💡 Promise 解决了回调地狱问题，使异步代码更易读，配合 Fetch 成为现代Web开发标配。
