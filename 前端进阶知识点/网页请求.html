📦 动态加载公共组件（今日所学）

成功加载 & 生命周期
// 从 API 获取数据并构造组件（模拟加载 header.html） fetch('https://jsonplaceholder.typicode.com/posts/1') .then(res => res.json()) .then(data => { const html = `<header style="background:#f1f5f9; padding:1rem; border-radius:8px;"> <h3>${data.title}</h3> <p>动态加载的组件内容</p> </header>`; document.getElementById('dynamic-header-placeholder').innerHTML = html; // 依赖 DOM 的代码放在这里执行 document.querySelector('#dynamic-header-placeholder button')?.addEventListener('click', () => alert('按钮被点击')); }) .catch(err => console.error(err));

成功加载后，在 .then() 中操作 DOM，确保组件已插入。

错误处理 & 友好提示
// 请求不存在的文件（header.html） fetch('header.html') .then(res => { if (!res.ok) throw new Error('网络响应失败'); return res.text(); }) .then(html => { document.getElementById('error-demo').innerHTML = html; }) .catch(error => { document.getElementById('error-demo').innerHTML = '<div class="error-box">❌ 加载失败：' + error.message + '，请稍后重试。</div>'; });

使用 .catch() 显示友好提示，避免页面空白。

原始代码：加载 header.html 到 <header>
// 您提供的原始代码（稍作适配，使用 #demo-header 演示） fetch('header.html') // 确保 header.html 位于网站根目录 .then(response => { if (!response.ok) { throw new Error(`HTTP 错误 ${response.status}`); } return response.text(); }) .then(html => { // 将获取到的 header 内容放入 <header> 标签 document.querySelector('#demo-header').innerHTML = html; // 可选：触发自定义事件 }) .catch(error => { console.error('加载 header 失败:', error); document.querySelector('#demo-header').innerHTML = '<div style="padding:1rem; background:#f0f0f0; text-align:center;">导航栏加载失败，请刷新重试</div>'; });
演示用 <header> 元素：
默认 header 内容（等待加载）

成功按钮使用 data: URL 模拟返回 HTML 片段；失败按钮请求不存在的 header.html。
观察上方 <header> 内容的变化及错误提示。

♿ 可访问性增强（今日所学）

键盘导航 :focus
<button class="focus-demo-btn">按钮1</button> <button class="focus-demo-btn">按钮2</button> <a href="#" class="focus-demo-btn">链接</a>
链接

按 Tab 键切换焦点，观察蓝色外圈样式。:focus 确保键盘用户能看到当前位置。

屏幕阅读器支持
<button aria-label="关闭对话框" onclick="alert('关闭')">X</button> <div role="alert">重要提示</div> <article> <h2>文章标题</h2> <p>内容...</p> </article>
⚠️ 重要提示（role="alert"）

语义化文章标题

使用 <article> 标签，屏幕阅读器可识别为独立内容。

添加 aria-label、role 和语义标签，提升辅助技术体验。

颜色对比度
<button class="contrast-bad">对比度不足</button> <button class="contrast-good">对比度良好</button>

左侧按钮文字与背景对比度低（浅灰/浅灰），难以阅读；右侧白色文字配深色背景，符合WCAG要求。
