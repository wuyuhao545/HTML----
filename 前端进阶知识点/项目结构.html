🧩 模块化开发：抽离公共组件

header.html (独立文件)
<!-- header.html --> <header class="site-header"> <img src="/picture/logo.png" alt="Logo"> <nav> <a href="/">首页</a> <a href="/about">关于</a> </nav> </header>

✔️ 一处编写，全局复用
✔️ 修改只需更新一个文件

fetch 加载组件
fetch('/header.html') .then(res => res.text()) .then(html => { document.getElementById('header-placeholder').innerHTML = html; // 在这里初始化交互 }) .catch(err => { console.error('加载失败', err); });

实际项目中，header.html 需放在服务器根目录，并用绝对路径引用内部资源。

模块化优势
  • ✅ 代码复用，减少重复
  • ✅ 维护简单，一处修改处处生效
  • ✅ 协作开发，分工明确
  • ✅ 缓存优化，公共文件可单独缓存

📌 变量提升与闭包 (Hoisting & Closure)

函数声明提升
// 可以在声明前调用 sayHi(); function sayHi() { return "Hello"; }
var 变量提升
console.log(a); // undefined var a = 5; console.log(a); // 5
let/const 暂时性死区
// 不能提前访问 console.log(b); // ReferenceError let b = 10;
闭包基本示例
function outer(x) { return function(y) { return x + y; }; } const add5 = outer(5);
内部函数记住外部作用域的变量。
计数器闭包
function createCounter() { let count = 0; return function() { return ++count; }; } const counter = createCounter();

🎨 HTML/CSS 组件化与模板化

<template> 标签
<!-- 定义模板（不可见） --> <template id="user-card"> <div class="card"> <h3>用户名</h3> <p>简介内容</p> </div> </template> <script> const template = document.getElementById('user-card'); const clone = template.content.cloneNode(true); container.appendChild(clone); </script>

<template> 内容不可见，可被 JS 克隆多次，实现组件复用。

CSS BEM 方法论
/* 块 Block */ .card { } /* 元素 Element */ .card__title { } .card__desc { } /* 修饰符 Modifier */ .card__desc--highlight { }

高亮描述（修饰符）

BEM 通过命名约定实现组件样式隔离，避免冲突。

Web Components 基础
// 定义自定义元素 class MyButton extends HTMLElement { constructor() { super(); this.attachShadow({ mode: 'open' }); this.shadowRoot.innerHTML = ` <button style="background:#3b82f6; color:white; border:none; padding:0.5rem 1rem; border-radius:30px;"> <slot>默认按钮</slot> </button> `; } } customElements.define('my-button', MyButton);

使用 Shadow DOM 封装内部结构和样式，生成可复用标签。

样式隔离 (Shadow DOM)
<style> /* 外部样式不影响 Shadow DOM 内部 */ .shadow-demo p { color: red; } </style> <div class="shadow-demo"> <p>外部普通段落（红色）</p> </div> <my-element></my-element>

外部段落（受全局样式影响，红色）

Shadow DOM 内部的元素不受外部 CSS 影响，实现真正隔离。
